Cordova Kitchen Sink

This is a Cordova application that demonstrates how to use each of the Apache Cordova Core APIs.

  • Battery Status
  • Camera
  • Console
  • Contacts
  • Device
  • Device Motion (Accelerometer)
  • Device Orientation (Compass)
  • Dialogs
  • FileSystem
  • File Transfer
  • Geolocation
  • Globalization
  • InAppBrowser
  • Media
  • Media Capture
  • Network Information (Connection)
  • Splashscreen
  • StatusBar
  • Vibration
